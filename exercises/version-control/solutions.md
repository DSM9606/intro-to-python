
# Version Control Exercise Solutions

Command-line approach:

```sh
# (create the remote repo and obtain its remote address - either the HTTPS or the SSH version)
cd ~/Desktop
git clone REMOTE_ADDRESS
cd my-first-repo/
code .

# (edit and save the files)
git add .
git commit -m "My first commit"
git push origin master

# (edit and save the files)
git add .
git commit -m "My second commit"
git push origin master

# (edit and save the files)
git add .
git commit -m "My third commit"
git push origin master
```
